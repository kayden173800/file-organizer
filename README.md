# File Organizer

A beginner Python automation project that automatically organizes files inside the Downloads folder into categorized folders.

## Features

* Organizes files automatically
* Creates folders if they do not exist
* Supports:

  * Images
  * PDFs
  * Videos
  * Documents
  * ZIP files
* Includes error handling

## Technologies Used

* Python
* pathlib
* shutil

## How To Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/file-organizer.git
```

2. Open the project folder

```bash
cd file-organizer
```

3. Run the script

```bash
python organizer.py
```

## Example

Before:

Downloads/

After:
